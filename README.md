# BitMemo (像素记) 📝

![Version](https://img.shields.io/badge/version-1.7.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)
![Style](https://img.shields.io/badge/style-8--bit%20Pixel-orange)

**BitMemo (像素记)** 是一款极具复古 8-bit 像素画风的浏览器便签扩展。它以“轻量、灵动、极客”为核心，为你提供一个随处可见、一触即发的灵感捕捉工具。

---

## ✨ 核心特性

- **像素美学 (Pixel Style)**：全界面采用经典的 8-bit 复古画风，硬核投影与粗边框，带你重回极客时代。
- **智能悬浮 (Floating Widget)**：页面侧边的小圆点是你的灵感入口。支持拖拽定位，双击即可展开面板。
- **多样化记录**：支持 **文本**、**链接** 与 **代码片段** 的分类记录。
- **智能开链 (Smart Link)**：自动识别 URL 并提供一键跳转预览。
- **自适应主题**：完美支持系统级的深色 (Dark) 与浅色 (Light) 模式切换。
- **本地存储**：数据归你所有，基于 `chrome.storage.local` 实现安全持久化。

## 🚀 快速开始

### 安装说明
1. 下载并解压本项目。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解压的扩展程序”，选择本项目文件夹即可。

### 交互指南
- **单次点击/拖拽**：移动悬浮球位置。
- **双击悬浮球**：展开或隐藏便签面板。
- **主页面开关**：在插件 Popup 页面顶部可根据需要彻底关闭网页悬浮球。
