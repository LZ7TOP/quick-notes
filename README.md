# Quick Notes LZ7 浏览器插件

一个极致简约、美观且实用的浏览器快捷便签插件。支持记录网址、文本和代码片段，并内置基础的代码格式化功能。

## ✨ 特性

- **分类管理**：支持“文本”、“链接”和“代码”三种类型的笔记，自动分类，清晰直观。
- **极致设计**：采用现代化的深色模式与毛玻璃（Glassmorphism）视觉效果。
- **搜索功能**：支持关键词实时检索，快速找到您的记录。
- **代码高亮与格式化**：为代码片段提供独立的显示样式，并支持一键缩进格式化。
- **本地存储**：数据归您所有，通过 `chrome.storage.local` 安全存储在您的浏览器中。

## 🚀 安装方法

1. 下载本项目代码到本地。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的 **“开发者模式” (Developer mode)**。
4. 点击左上角的 **“加载已解压的扩展程序” (Load unpacked)**。
5. 选择本项目文件夹即可完成安装。

## 🛠️ 技术实现

- **Manifest V3** 标准。
- **Vanilla JS & CSS**：无外部依赖，极速加载。
- **Chrome Storage API**：实现持久化存储。
