## 中文

**BitMemo (像素记)** 是一款极具复古 8-bit 像素画风的浏览器便签扩展。它以“轻量、灵动、极客”为核心，为你提供一个随处可见、一触即发的灵感捕捉工具。

### ✨ 核心特性

- **像素美学 (Pixel Style)**：全界面采用经典的 8-bit 复古画风，硬核投影与粗边框，带你重回极客时代。
- **智能悬浮 (Floating Widget)**：页面侧边的小圆点是你的灵感入口。支持拖拽定位，双击即可展开面板。
- **多端实时同步**：浏览器主弹窗与网页悬浮窗之间数据即时同步，无缝衔接。
- **多样化记录**：支持 **文本**、**链接** 与 **代码片段** 的分类记录。
- **智能开链 (Smart Link)**：自动识别 URL 并提供一键跳转预览。
- **自适应主题**：完美支持系统级的深色 (Dark) 与浅色 (Light) 模式切换。
- **本地存储**：数据归你所有，基于 `chrome.storage.local` 实现安全持久化。

### 🚀 快速开始

#### 安装说明
1. 下载并解压本项目。
2. 打开 Chrome 浏览器，进入 `chrome://extensions/`。
3. 开启右上角的“开发者模式”。
4. 点击“加载已解解压的扩展程序”，选择本项目文件夹即可。

#### 交互指南
- **单次点击/拖拽**：移动悬浮球位置。
- **双击悬浮球**：展开或隐藏便签面板。
- **主页面开关**：在插件 Popup 页面顶部可根据需要彻底关闭网页悬浮球。

## English

**BitMemo** is a browser extension for quick notes featuring a retro 8-bit pixel art style. Centered around "Lightweight, Dynamic, and Geeky" values, it provides you with an ever-present tool to capture inspirations instantly.

### ✨ Key Features

- **Pixelized Aesthetics**: The entire interface adopts a classic 8-bit retro style with hard shadows and thick borders, bringing you back to the golden age of geeks.
- **Floating Widget**: A small draggable bubble on the side of your web page serves as your entry point for ideas. Supports double-click to expand/collapse.
- **Real-time Sync**: Instant data synchronization between the main extension popup and the in-page floating panel.
- **Versatile Records**: Categorized support for **Text**, **Web Links**, and **Code Snippets**.
- **Smart Link Activation**: Automatically detects URLs and provides one-click navigation.
- **Adaptive Theme**: Fully supports system-level Dark and Light mode switching.
- **Local Storage**: Your data belongs to you, securely persisted via `chrome.storage.local`.

### 🚀 Quick Start

#### Installation
1. Download and extract this repository.
2. Open Chrome and navigate to `chrome://extensions/`.
3. Enable "Developer mode" in the top right corner.
4. Click "Load unpacked" and select the project folder.

#### Interaction Guide
- **Single Click / Drag**: Move the floating widget position.
- **Double Click**: Expand or hide the note panel.
- **Global Toggle**: Use the switch at the top of the extension popup to completely enable or disable the floating widget.
