# Changelog

## [1.1.0] - 2026-03-20

### ✨ 新特性

- **浮窗定位 (Draggable Widget)**：在所有网页中引入了可拖拽的悬浮按钮与面板，支持随处开启便签。
- **一键开链 (Smart Link)**：针对链接类型的便签新增了“打开”按钮，支持在新标签页一键跳转。
- **UI 深度优化**：优化了毛玻璃视觉深度，增强了卡片悬浮时的阴影与过渡动画反馈。

### 🛠️ 优化

- **架构升级**：引入了 Background Service Worker 与 Content Scripts，提升了扩展的稳定性和跨页面响应速度。
- **交互提升**：支持自动识别链接内容并补充默认标题。

## [1.0.0] - 2026-03-20

### ✨ 新特性

- **核心功能**：支持记录“文本”、“链接”和“代码”三种类型的便签。
- **搜索系统**：实时关键词搜索，快速检索所有记录。
- **代码增强**：内置单层缩进的代码格式化功能，并适配 `JetBrains Mono` 代码字体。
- **现代化 UI**：全新的毛玻璃（Glassmorphism）视觉风格，支持深色沉浸式体验。
- **本地存储**：基于 `chrome.storage.local` 的安全持久化方案。

### 🛠️ 优化

- 优化了弹出窗口的加载速度。
- 改进了分类选项卡的切换动画。

### 🐞 修复

- 修复了 Manifest V3 下的一些权限警告问题。
