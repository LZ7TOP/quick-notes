# Changelog

## [2.4.1] - 2026-03-20

### 🩹 紧急修复

- **语法错误修复**：修复了 `js/content.js` 中由于缺少关闭闭包括号导致的 `SyntaxError`，确保悬浮窗和右键反馈功能在所有页面正常运行。

## [2.4.0] - 2026-03-20

### 🧠 智能识别与来源追踪

- **智能类型识别**：在手动输入或通过右键菜单采集时，系统会自动分析内容并智能判定为“文本”、“链接”或“代码”。在手动新建时，类型选择将随输入实时自动切换。
- **标注来源属性**：为每条便签新增了“来源”标签（手动 / 右键），帮助用户清晰区分采集途径。
- **多语言全面同步**：完成了中（简/繁）、英、日、韩五种语言下所有新增功能的国际化配置补全。

## [2.3.0] - 2026-03-20

### ✨ 快捷采集与交互增强

- **右键菜单支持**：新增了全局右键菜单。现在可以在网页中直接选中文字或右键点击链接，选择“添加到 像素记”快速保存记录。
- **采集反馈提示**：在网页中通过右键菜单添加记录后，会自动弹出沉浸式的像素风提示，无需切换页面即可确认添加成功。
- **全语种同步**：右键菜单名称已完美适配中、繁、英、日、韩五种语言。

## [2.2.0] - 2026-03-20

### ✨ 功能新增与国际化补全

- **数据一键清空**：新增了清空全部便签的功能，并配备了像素风的二次确认弹窗，防止误操作。
- **全语种配置同步**：补全了英语、日语、韩语及繁体中文下的所有新增功能翻译项（包括复制成功提示和清空确认提示）。

## [2.1.2] - 2026-03-20

### 🩹 细节修复与体验优化

- **图标展示修复**：修复了图标虽然为 `.png` 后缀但内部数据为 `JPEG` 导致浏览器无法识别的问题。现在所有图标均为标准的 PNG 格式。
- **自定义复制反馈**：新增了“复制成功”的弹出提示 (Toast)，支持五国语言自适应显示，提升了交互确认感。

## [2.1.0] - 2026-03-20

### 🌍 多语言支持 (Internalized & Globalized)

- **五国语言适配**：新增对简体中文、繁体中文、英语、日语、韩语的全面支持。
- **i18n 引擎**：构建了轻量级国际化方案，支持平滑切换语言而无需重启扩展。
- **语言持久化**：用户选择的语言会自动保存并跨终端同步。
- **剪贴板权限修复**：解决了网页悬浮窗内无法使用复制功能的问题，通过授权 iframe 剪贴板写入权限修复了安全策略报错。

## [1.9.0] - 2026-03-20

### ✨ 数据同步与组件升级

- **实时数据同步**：引入了存储监听机制 (`chrome.storage.onChanged`)，实现了浏览器主弹窗与网页悬浮窗之间的数据实时同步。现在在任何一端修改便签，另一端都会立即自动刷新。
- **自定义像素下拉框**：彻底废弃了原生 `select` 标签，改为使用纯 `div` 构建的像素风格下拉组件，解决了不同操作系统下样式不统一的问题，完美适配 8-bit 复古视觉。
- **尺寸与布局优化**：将悬浮窗面板高度固定为 600px，并修复了 GitHub 链接在部分场景下不可见的问题。

## [1.8.0] - 2026-03-20 (Internal)

### ✨ UI 与体验初步优化

- **悬浮窗尺寸适配**：初步调整高度检测。
- **自定义下拉框样式**：Native select 样式改良。

## [1.7.0] - 2026-03-20

### ✨ 品牌重塑 (Rebranding)

- **正式更名**：项目正式由 "Quick Notes" 更名为 **BitMemo (像素记)**。
- **UI 更新**：更新了插件弹出页面的标题和页眉，统一了品牌视觉。
- **文档同步**：全面更新了 `README.md` 和项目元数据，开启 1.7.x 时代。

## [1.6.2] - 2026-03-20

### 🚀 发布流程修复

- **发布说明自动化**：优化了 GitHub Actions 发布脚本，现在 GitHub 的 Release 页面将只显示当前版本的更新内容，而不再显示整个变更历史。
- **打包路径修复**：修正了自动打包脚本，确保在 v1.6.0 目录重构后的资源（`/css`、`/js`、`/assets`）能被正确包含在 ZIP 包中。

## [1.6.1] - 2026-03-20

### 🐞 修复与清理

- **紧急修复**：修复了 v1.6.0 版本中由于重构导致的悬浮窗无法展开（透明度保持为 0）的严重 Bug。
- **日志清理**：优化了更新日志的编写逻辑，去除了多余的重复内容。

## [1.6.0] - 2026-03-20

### 📦 性能与结构优化

- **资源本地化**：将 `DotGothic16` 字体文件直接集成到项目中（`assets/fonts/`），去除了对 Google Fonts 外部链接的依赖，提升了加载速度并确保在离线状态下也能正常显示像素字体。
- **目录结构重构**：优化了项目文件组织方式，按类型划分为 `/css`、`/js` 和 `/assets` 目录，使项目结构更加专业、清晰且易于维护。
- **配置同步更新**：全面更新了 `manifest.json`、`popup.html` 以及 `content.js` 中的资源引用路径，确保在新的目录结构下各项功能运行正常。

## [1.5.0] - 2026-03-20

### ✨ 功能与交互优化

- **新增设置选项**：在主页面顶部增加了“显示悬浮窗”开关，用户可以根据喜好随时启用或禁用页面上的悬浮球。
- **视觉协调性提升**：
  - 为面板展开和关闭增加了平滑的谈入淡出动画，视觉体验更连贯。
  - **智能关闭按钮**：关闭 `[x]` 按钮现在会根据面板的展开方向自动定位到最外侧的角落，避免与悬浮按钮重叠，且更符合操作直觉。
  - 增强了悬浮球在屏幕边缘时的定位稳定性，解决了展开时可能产生的跳变感。

## [1.4.0] - 2026-03-20

### ✨ 交互优化

- **激活方式变更**：悬浮按钮现在改为 **双击 (Double Click)** 展开或收起便签面板，单击和长按拖拽现在完全用于移动位置，极大地减少了移动时的误触。
- **智能方位适配**：面板展开时会根据当前悬浮按钮在屏幕中的位置（四个象限），自动调整展开方向（向上/下、向左/右），确保面板始终完整显示在屏幕范围内。

## [1.3.0] - 2026-03-20

### ✨ [1.3.0] 新特性与优化

- **自适应主题 (Adaptive Theme)**：支持根据浏览器/系统设置自动切换深色和浅色模式。
- **便签编辑功能**：新增编辑按钮，支持对已有的便签内容进行重新编辑和保存。
- **滚动内容支持**：优化了便签卡片高度，当内容较长时支持卡片内滚动，并适配了像素风格滚动条。

## [1.2.0] - 2026-03-20

### ✨ [1.2.0] 新特性与优化

- **像素画风 UI (Pixel Style)**：全面重构了扩展的视觉设计，移除了毛玻璃效果，采用经典的 8-bit 复古像素风格。更新了字体（引入 `DotGothic16` 与等宽字体 fallback），界面增加了硬核阴影与粗黑边框。
- **悬浮组件交互增强**：彻底修复了网页中悬浮按钮在拖拽和点击时容易发生误触从而意外展开面板的问题。
- **关闭按钮**：为浮窗面板增加了明显的 `[x]` 关闭按钮，解决了悬浮窗无法直观关闭的困扰。

## [1.1.0] - 2026-03-20

### ✨ [1.1.0] 新特性

- **浮窗定位 (Draggable Widget)**：在所有网页中引入了可拖拽的悬浮按钮与面板，支持随处开启便签。
- **一键开链 (Smart Link)**：针对链接类型的便签新增了“打开”按钮，支持在新标签页一键跳转。
- **UI 深度优化**：优化了毛玻璃视觉深度，增强了卡片悬浮时的阴影与过渡动画反馈。

### 🛠️ 优化

- **架构升级**：引入了 Background Service Worker 与 Content Scripts，提升了扩展的稳定性和跨页面响应速度。
- **交互提升**：支持自动识别链接内容并补充默认标题。

## [1.0.0] - 2026-03-20

### ✨ [1.0.0] 新特性

- **核心功能**：支持记录“文本”、“链接”和“代码”三种类型的便签。
- **搜索系统**：实时关键词搜索，快速检索所有记录。
- **代码增强**：内置单层缩进的代码格式化功能，并适配 `JetBrains Mono` 代码字体。
- **现代化 UI**：全新的毛玻璃（Glassmorphism）视觉风格，支持深色沉浸式体验。
- **本地存储**：基于 `chrome.storage.local` 的安全持久化方案。

### 🛠️ 优化

- 优化了弹出窗口的加载速度。
- 改进了分类选项卡的切换动画。

### 🐞 修复

- 修复了 Manifest V3 下的一些权限警告问题。
